# Web UI Bundles

Web UI bundles for [Gestalt](https://github.com/valon-technologies/gestalt).
Each bundle packages a frontend application that `gestaltd` serves to users.

See the [configuration guide](https://gestaltd.ai/configuration) for setting up
web UI providers and the
[provider development guide](https://gestaltd.ai/providers) for writing custom
web bundles.

## Available Providers

| Provider | Version | Description |
|----------|---------|-------------|
| [Default Web UI](default/) | `0.0.1-alpha.9` | Default Gestalt web UI bundle served at / |
