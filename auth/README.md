# Identity Providers

Identity providers for [Gestalt](https://github.com/valon-technologies/gestalt).
Each provider implements an identity provider that `gestaltd` uses to verify
user identity and manage sessions.

See the [configuration guide](https://gestaltd.ai/configuration) for setting up
identity providers and the
[provider development guide](https://gestaltd.ai/providers) for writing custom
identity providers.
